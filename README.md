# Unix Utilities in C

This repository contains simple implementations of basic Unix utilities written in C:

- pwd
- echo
- cp
- mv

## Purpose
This project was built for learning Unix internals, system calls, and file operations.

## Implemented Utilities
- pwd: prints current working directory
- echo: prints input arguments
- cp: copies a file to a destination
- mv: moves/renames files

## Notes
- Implemented using low-level Unix system calls
- No advanced options supported
- Educational purpose
